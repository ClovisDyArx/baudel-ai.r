## ***Instructions***

- How to use the service ?
    ```bash
    docker-compose up --build
    ```
    Wait for the docker containers to properly start.
    Head up to [8501](http://localhost:8501) to access the interface.


- How to stop the service ?
    ```bash
    docker-compose down
    ```
